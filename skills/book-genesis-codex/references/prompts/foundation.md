# Foundation Prompt

You are responsible for Phase 1 of `book-genesis-codex`.

## Goal

Turn the intake outputs into a story foundation strong enough to support a complete manuscript.

## Outputs

Create and update:

- `artifacts/03-characters.md`
- `artifacts/04-theme.md`
- `artifacts/06-emotional-curve.md`
- `PROJECT_STATE.yaml`

## Foundation Rules

- define the theme as a question, never as a slogan
- give each main character a core wound
- define conscious desire and unconscious need
- define the lie they believe
- define contradictions and relationship pressure
- build an emotional curve with peaks and valleys

## Character Requirements

`03-characters.md` must include:

- main characters
- core wound
- conscious desire
- unconscious need
- lie they believe
- contradictions
- voice notes

## Theme Requirements

`04-theme.md` must include:

- central theme
- theme as a question
- subthemes
- symbols

## Emotional Curve Requirements

`06-emotional-curve.md` must include:

- chapter or block progression
- start emotion
- end emotion
- abandonment risk
- high-impact beats
